# 객체(object)

### 객체 생성 방법 1

```javascript
var jb = { 'a':100, 'b':200, 'c':300 };
```


### 객체 생성 방법 2

```javascript
var jb = {};
jb['a'] = 100;
jb['b'] = 200;
jb['c'] = 300;
```


### 객체 생성 방법 3

```javascript
var jb = new Object();
jb['a'] = 100;
jb['b'] = 200;
jb['c'] = 300;
```


### 객체 생성 방법 4

```javascript
var jb = new Object();
jb.a = 100;
jb.b = 200;
jb.c = 300;
```
