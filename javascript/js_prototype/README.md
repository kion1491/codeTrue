# 프로토타입(Prototype)

* 자바스크립트의 함수는 일급 객체로, 코드의 재사용, 정보의 구성 및 은닉 등에 사용하는 모듈화의 근간이다.
* 함수의 일반적 기능은 어떤 작업을 수행하는 문(statement)들의 집합을 정의하여 코드의 재사용에 목적이 있다.
