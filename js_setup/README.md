# 자바스크립트 동작 원리

* 모든 브라우저는 자바스크립트를 해석하고 실행할 수 있는 **자바스크립트 엔진**을 내장하고 있습니다.  
* 브라우저뿐만 아니라 Node.js도 자바스크립트 엔진을 내장하고 있습니다.  
* 기본적으로 브라우저에서 동작하는 코드는 Node.js 환경에서도 동작합니다.
